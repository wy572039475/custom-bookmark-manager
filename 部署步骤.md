## 🎯 快速部署步骤

### 第 1 步：部署后端到 Render (10分钟)

1. 访问 https://dashboard.render.com
2. 点击 "New" → "Web Service"
3. 连接 GitHub 仓库 `custom-bookmark-manager`
4. 配置基本信息:
   - Name: `custom-bookmark-backend`
   - Region: `Oregon (US West)` 或 `Singapore`
   - Branch: `main`
   - Root Directory: `server`
   - Build Command: `npm install && npm run build`
   - Start Command: `npm run db:migrate:turso && npm start`

5. 添加环境变量
   - `TURSO_DATABASE_URL` - 你的 Turso 数据库 URL
   - `TURSO_AUTH_TOKEN` - 你的 Turso Auth Token
   - `JWT_SECRET` - 点击 "Generate" 生成随机密钥
   - `JWT_EXPIRES_IN=7d`
   - `CORS_ORIGIN` - 随意填写(部署前端后更新)
   - `USE_LOCAL_SQLITE=false`

6. 点击 "Create Web Service"

### 第 2 步:部署前端到 Vercel (5分钟)

1. 访问 https://vercel.com
2. 点击 "Add New" → "Project"
3. 导入 GitHub 仓库 `custom-bookmark-manager`
4. 配置基本信息
   - Project Name: `custom-bookmark-manager`
   - Framework Preset: `Vite`
   - Root Directory: `client`
   - Build Command: `npm run build`
   - Output Directory: `dist`

5. 添加环境变量
   - `VITE_API_URL=https://你的后端URL.onrender.com/api

6. 点击 "Deploy"

### 第 3 步:更新 CORS (2分钟)
1. 回到 Render 后端设置
2. 更新环境变量 `CORS_ORIGIN` 为你的前端 URL

3. Render 会自动重启
