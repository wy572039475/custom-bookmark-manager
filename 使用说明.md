# 工作导航 - 使用说明

## 问题已解决

如果界面显示空白，现在系统会自动重定向到登录页面。

## 快速开始

### 1. 访问应用
打开浏览器访问: `http://localhost:5173/`

### 2. 登录或注册
- 如果已有账号，点击"登录"按钮
- 如果是新用户，点击"立即注册"

### 3. 模拟模式（推荐用于测试）
系统会在后端不可用时自动切换到模拟模式（离线模式）。

**模拟模式特点：**
- 无需配置数据库
- 数据保存在浏览器本地存储中
- 支持完整的收藏管理功能
- 可以导出和导入数据

### 4. 开始使用
登录后，您可以：
- ✅ 添加收藏（点击"添加收藏"按钮）
- ✅ 创建分类（点击"添加分类"按钮）
- ✅ 搜索收藏（在搜索框输入关键词）
- ✅ 按分类筛选（点击左侧分类）
- ✅ 编辑和删除收藏
- ✅ 导出数据（点击"导出"按钮）
- ✅ 导入数据（点击"导入"按钮）

## 功能说明

### 离线模式标识
当使用模拟模式时，页面右上角会显示黄色的"离线模式"标签。

### 清除数据
在离线模式下，左侧侧边栏底部有"清除本地数据"按钮，可以删除所有本地存储的数据。

## 故障排除

### 问题：页面仍然空白
**解决方法：**
1. 检查浏览器控制台是否有错误（按F12打开开发者工具）
2. 刷新页面（Ctrl+F5）
3. 清除浏览器缓存
4. 确保开发服务器正在运行

### 问题：无法登录
**解决方法：**
1. 如果使用真实后端，确保后端服务器正在运行（端口3001）
2. 如果后端不可用，系统会自动切换到模拟模式
3. 尝试注册新账号

### 问题：数据丢失
**解决方法：**
- 在模拟模式下，数据保存在浏览器本地存储中
- 定期使用"导出"功能备份数据
- 清除浏览器缓存会导致模拟数据丢失

## 开发模式

### 当前运行状态
- 前端服务器: http://localhost:5173/
- 后端API: http://127.0.0.1:3001/api

### 停止服务器
按 `Ctrl+C` 停止运行中的服务器

### 重新启动
在项目根目录运行:
```bash
npm run dev
```

## 文件说明

- `fullstack-demo.html` - 完整的HTML演示版本（无需服务器）
- `fullstack-production.html` - 生产版本HTML
- `bookmark-offline.html` - 离线版本
- `client/` - React前端应用
- `server/` - Express后端服务器

## 技术栈

- 前端: React 18 + TypeScript + Vite + Tailwind CSS
- 后端: Express + TypeScript
- 数据库: PostgreSQL（可选，支持模拟模式）

## 支持

如有问题，请查看浏览器控制台的错误信息。
